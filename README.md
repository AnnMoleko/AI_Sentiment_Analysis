# The Impact of AI on Society — Sentiment Analysis

This project analyzes public opinions on the impact of Artificial Intelligence using sentiment analysis.

### Tools Used
- Excel (Data Cleaning)
- Python (TextBlob for Sentiment Analysis)
- Power BI (Visualization)

### Folder Structure
- `data/` – Raw and processed data
- `notebooks/` – Python sentiment analysis notebook
- `visuals/` – Charts and Power BI dashboard
